# 在服务端渲染React组件（hello world）

操作步骤：

1. 编写一个React组件
2. 搭建一个Node服务（Koa）
3. 使用react-dom/server的renderToString方法将React组件渲染成字符串（data-reactroot）。
4. 编译代码（babel）
5. 运行服务


启动过程：

```
// 1. 安装依赖包
yarn

// 2. babel编译代码

yarn run complied

// 3. 运行服务

yarn run start

// 4. 打开浏览器

http://localhost:9001
```
